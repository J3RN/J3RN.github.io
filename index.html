Web Technologies

HTML5
CSS3
jQuery
PHP

HTML5 is the latest version of the HyperText Markup Language. HTML5 supports many features that formerly required plugins like Flash, such as audio clips, video, and many others!

CSS3 is the third rendition of Cascading Style Sheets, the language that makes web pages beautiful! CSS3 supports, among other things, animations, text effects, and on-mouse-over actions.

jQuery's motto is "Write Less, Do More." It enables you to do just that! jQuery is a Javascript library that makes your code beautiful and does all of the heavy lifting for you!

PHP is one of the most common scripting languages of the web. It does much of the complex functions of web apps, and is the backbone of sites like Facebook, Photobucket, and Wikipedia.

App Technologies

Windows 8
Android

Windows 8 is the latest version in the Windows line of operating systems from Microsoft. Somewhat due to it's slow adoption, Microsoft has made it easy for developers to design what used to be called "Metro" apps for the operating system.

Android, at the time of this writing, is the most popular mobile operating system. Android is an open source project, backed by Google. Due to its open source nature, learning to develop for Android is very easy.
